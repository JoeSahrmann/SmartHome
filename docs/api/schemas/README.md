# SmartHome API Schemas

Machine-readable API contract schemas are versioned by folder.

- `v1/` contains pre-implementation JSON Schema contract fragments for:
  - `bgl`, `sgl`, `cco`, `cm`, `ch`, `hm`, `hbf`
  - shared definitions in `common.v1.json`

These are intended as source-of-truth artifacts alongside `docs/api/contracts.md`.
